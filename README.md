# ToDoアプリ

仕事や日常のやることを管理するための、Windows デスクトップアプリケーションです。

---

## 動作環境

- Windows
- Python 3.x
- オフライン利用可能

---

## セットアップ

初回のみ、以下の手順でセットアップを行ってください。

1. `run_setup.bat` をダブルクリックして実行します。
   - Python の仮想環境（venv）が作成されます。
   - 必要なパッケージが自動的にインストールされます。

---

## 起動方法

セットアップ完了後、以下の手順でアプリを起動します。

1. `run_app.bat` をダブルクリックして実行します。

---

## 機能一覧

### ホーム画面

アプリ起動時に表示される画面です。以下のタスクを確認できます。

| セクション | 内容 |
| --- | --- |
| 期限切れタスク | 期限が過ぎた未完了タスク |
| 今日期限タスク | 本日が期限の未完了タスク |
| 重要タスク | 優先度が高い未完了タスク |

### タスク一覧画面

すべての未完了タスクを一覧表示します。

- タスクの追加・編集・削除
- チェックボックスでタスクを完了にする
- 完了したタスクは自動的にアーカイブへ移動

**表示順：** 優先度（高い順）→ 期限（近い順）→ 作成日（古い順）

### アーカイブ画面

完了済みタスクの一覧を表示します。

- アーカイブ済みタスクを未完了状態に戻す（復元）
- アーカイブ済みタスクを削除

---

## タスクの登録項目

| 項目 | 必須 | 説明 |
| --- | --- | --- |
| タイトル | ✅ | タスク名 |
| 期限 | ー | 未設定も可 |
| タグ | ー | 下記の固定タグから選択 |
| 優先度 | ー | 5段階で設定（初期値：★★★☆☆） |

### タグ

以下の固定タグから選択できます。

- 仕事
- 個人
- 開発
- 勉強
- 買い物
- その他

### 優先度

5段階で設定できます。

| 優先度 |
| --- |
| ★★★★★ |
| ★★★★☆ |
| ★★★☆☆（初期値） |
| ★★☆☆☆ |
| ★☆☆☆☆ |

---

## 通知機能

アプリ起動時に本日期限のタスクがある場合、Windows 通知が表示されます。

```
今日期限のタスクがあります

・会議資料作成
・顧客メール返信
```
